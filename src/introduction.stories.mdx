# Welcome to Ceramic

Ceramic is a reusable React component library.

Browse the available components now by navigating to them in the sidebar.

## Want to use Ceramic Components in your project?

Start by adding Ceramic as a dependency:

```bash
yarn add ceramic-components
```

Or if you prefer NPM:

```bash
npm install ceramic-components
```

Then simply import components wherever you need them, for example:

```js
import { Join } from "ceramic-components";
```
